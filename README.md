# Aplicación Estadísticas PDI

Proyecto en Streamlit que permite visualizar y explorar estadísticas policiales de la Policía de Investigaciones de Chile.

## Estructura
- app.py — Código principal
- data/pdi_estadisticas.csv — Base de datos
- requirements.txt — Librerías necesarias

## Objetivo
Crear un dashboard interactivo y un buscador de estadísticas policiales.
